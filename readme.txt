    在实际工作中，我们常需要用到多台服务器，应用于分布式架构，可能会对服务器进行分组，并将服务器ip、port存储到一个文本文件中。
如：
[webservers]
192.168.203.128:22
192.168.203.129:22
192.168.203.130:22
[mysqlservers]
10.1.1.14:2222
10.1.1.15:22
......
这种格式

本python脚本主要用于文本过滤出你需要操作的哪个服务器组的主机ip、port,并连接到对应的主机，实现一条命令在多台同组的服务器上运行，
提供了windows向linux推送公钥的功能（用于免密），
并实现将文本的信息存储到数据库，可以直接从数据库中读取ip,连接到对应的主机，减少IO消耗
